Documentation
=============

Welcome to the :py:mod:`starry` documentation. The :py:mod:`starry` code
package is a suite of tools for mapping stars and exoplanets based on
timeseries data. Check out the links below for examples, tutorials, detailed
documentation of the API, and more.
