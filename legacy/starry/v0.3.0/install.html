Installation¶

Using conda¶

The easiest way to install starry is using conda via conda-forge. First, ensure you have the conda-forge channel enabled:

conda config --add channels conda-forge

Once that’s done, installing starry is easy:

conda install starry

Using pip¶

The next easiest way to install starry is using the pip command:

pip install starry

This will download, compile, and install starry. Depending on your machine, compiling may take a couple minutes. Once that’s done, open a python terminal, import starry, and hack away!

Development version¶

You can install the development version of starry directly from GitHub:

git clone https://github.com/rodluger/starry.git
cd starry
python setup.py develop
