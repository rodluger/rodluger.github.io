Positive semi-definite surface maps¶

Derivation of the positive semi-definite condtion for \(l = 1\) maps¶

If we denote our map coefficients by \(\mathbf{y} = \{ \mathbf{y_0}, \mathbf{y_1}, \mathbf{y_2}, \mathbf{y_3} \}\), the specific intensity of a map of degree \(l_\mathrm{max} = 1\) as a function of \(x\) and \(y\) is given by

:nbsphinx-math:`begin{align}
I(x, y) &= mathbf{y_0} Y_{0,0} + mathbf{y_1} Y_{1,-1} + mathbf{y_2} Y_{1,0} + mathbf{y_3} Y_{1,1} \
&= mathbf{y_0} sqrt{frac{1}{4pi}}

  • mathbf{y_1} sqrt{frac{3}{4pi}} y

  • mathbf{y_2} sqrt{frac{3}{4pi}} sqrt{1 - x^2 - y^2}

  • mathbf{y_3} sqrt{frac{3}{4pi}} x

end{align}`

To ensure the map is positive semi-definite everywhere, the minimum value of \(I(x, y)\) must be \(\ge 0\). We therefore differentiate it with respect to \(x\) and \(y\) and set the derivatives to zero to obtain two constraints on the location of the extrema:

:nbsphinx-math:`begin{align}

frac{mathrm{d}I}{mathrm{d}x} = sqrt{frac{3}{4pi}}left(-frac{mathbf{y_2} x}{sqrt{1 - x^2 - y^2}} + mathbf{y_3}right) = 0

end{align}`

and

:nbsphinx-math:`begin{align}

frac{mathrm{d}I}{mathrm{d}y} = sqrt{frac{3}{4pi}}left(-frac{mathbf{y_2} y}{sqrt{1 - x^2 - y^2}} + mathbf{y_1}right) = 0

end{align}`

so the values of \(x\) and \(y\) at the extrema satisfy

:nbsphinx-math:`begin{align}

tag{1} mathbf{y_2} x &= mathbf{y_3}sqrt{1 - x^2 - y^2}

end{align}`

and

:nbsphinx-math:`begin{align}

tag{2} mathbf{y_2} y = mathbf{y_1}sqrt{1 - x^2 - y^2}

end{align}`

Rearranging (1),

:nbsphinx-math:`begin{align}

x^2 &= frac{mathbf{y_3}^2}{(mathbf{y_2}^2 + mathbf{y_3}^2)}(1 - y^2).

end{align}`

Plugging into (2) and rearranging, we can solve for \(y\):

\begin{align} y = \pm \frac{\mathbf{y_1}}{\sqrt{\mathbf{y_1}^2 + \mathbf{y_2}^2 + \mathbf{y_3}^2}} \end{align}

and then for \(x\):

\begin{align} x = \pm \frac{\mathbf{y_3}}{\sqrt{\mathbf{y_1}^2 + \mathbf{y_2}^2 + \mathbf{y_3}^2}} \end{align}

Finally, plugging these back into the equation for \(I(x, y)\), we get the value of the specific intensity at the extrema:

:nbsphinx-math:`begin{align}
I(x, y) &= mathbf{y_0} sqrt{frac{1}{4pi}}

pm sqrt{frac{3}{4pi}} frac{mathbf{y_1^2}}{sqrt{mathbf{y_1}^2 + mathbf{y_2}^2 + mathbf{y_3}^2}} + mathbf{y_2} sqrt{frac{3}{4pi}} frac{mathbf{y_2^2}}{sqrt{mathbf{y_1}^2 + mathbf{y_2}^2 + mathbf{y_3}^2}} pm sqrt{frac{3}{4pi}} frac{mathbf{y_3^2}}{sqrt{mathbf{y_1}^2 + mathbf{y_2}^2 + mathbf{y_3}^2}} \ &= sqrt{frac{1}{4pi}} left( mathbf{y_0} pm sqrt{3} frac{mathbf{y_1}^2 + mathbf{y_2}^2 + mathbf{y_3}^2}{sqrt{mathbf{y_1}^2 + mathbf{y_2}^2 + mathbf{y_3}^2}} right)

end{align}`

By inspection, the minimum value corresponds to the solution with the minus sign. So the necessary and sufficient condition for \(I(x,y) \ge 0\) is

:nbsphinx-math:`begin{align}

mathbf{y_1}^2 + mathbf{y_2}^2 + mathbf{y_3}^2 le frac{mathbf{y_0}^2}{3}

end{align}`
